IPython & Jupyter Magic Cheatsheet

Line magics, cell magics, shell escapes, and notebook keyboard shortcuts
LINEworks on one line — %magic
CELLworks on whole cell — %%magic
KEYkeyboard shortcut

Essentials & Help

%quickrefOpen IPython's built-in quick reference card.
%magicPrint detailed docs for every available magic command.
%lsmagicList all currently available line and cell magics.
object?Show docstring / quick info for object. Use object?? to also show source code.
%pinfo objSame as obj? — detailed object information.
%who / %whosList variables in the interactive namespace; %whos adds type and value.
%resetClear all variables from the namespace. Add -f to skip the confirmation prompt.
%envList, get, or set environment variables, e.g. %env MY_VAR=value.
%configInspect or set configurable IPython class options.

Timing & Profiling

%time stmtTime a single execution of a statement (wall time + CPU time).
%%timeTime the execution of an entire cell.
%timeit stmtTime a statement over multiple runs/loops for a stable average.
%%timeitTime an entire cell over multiple runs. Use -n and -r to set loops/repeats.
%prun stmtRun statement under the Python profiler and show a function-call report.
%%prunProfile an entire cell's execution.
%memit stmtMeasure memory usage of a statement (requires memory_profiler).
%mprunLine-by-line memory profiler for a function (requires memory_profiler).

Debugging & Errors

%debugDrop into the interactive debugger at the point of the last exception.
%pdbToggle automatic debugger activation whenever an uncaught exception occurs.
%tbRe-print the last traceback without re-running anything.
%xmode modeSet traceback verbosity: Plain, Context, or Verbose.
%recall nRepeat the source for input history line n into the current prompt.
%macroSave a sequence of previous input lines as a reusable macro.

Running Code

%run file.pyExecute an external Python script inside the current namespace.
%load file.pyLoad a script's contents into the current cell for editing/running.
%load_ext nameLoad an IPython extension module by name.
%reload_ext nameReload a previously loaded extension (handy after editing it).
%%captureSuppress or capture a cell's stdout/stderr/display output into a variable.
%%scriptRun the cell using an external shell/script command, e.g. %%script bash.
%%writefile fWrite the cell's contents to a file. Add -a to append instead of overwrite.
%pycat fileShow a syntax-highlighted view of a Python file without executing it.

Other Languages & Shell

!cmdRun any shell command, e.g. !ls -la. Output can be captured into a variable.
!!cmdRun a shell command and return output as a list of lines.
%aliasDefine a shorthand alias for a system command.
%cd pathChange the current working directory (keeps a history with %cd -).
%pwdPrint the current working directory.
%%bashRun the entire cell as a Bash script.
%%HTMLRender the cell's contents directly as HTML in the notebook.
%%javascriptExecute the cell as JavaScript in the notebook front-end.
%%latexRender the cell's contents as LaTeX.
%%markdownRender the cell's contents as Markdown.
%%svgRender the cell's contents as an inline SVG image.
%%perl / %%rubyRun the cell using Perl or Ruby (if installed).

Plotting & Display

%matplotlib inlineEnable inline matplotlib plots rendered directly under cells.
%matplotlib widgetEnable interactive, pannable/zoomable matplotlib plots.
%pylabImport numpy and matplotlib's pyplot into the namespace (legacy; explicit imports preferred).
%precision nSet the number of decimal places used when displaying floats.
%%latexRender cell contents as typeset LaTeX math/text.

History & Variables

%historyShow input history. Add -n for line numbers, -o to include output.
_ / __ / ___Last, second-to-last, and third-to-last output values.
_i / _iiLast and second-to-last input strings (the code itself, not its result).
_n / Out[n]Output of execution number n, e.g. _7 or Out[7].
%storePersist a variable's value across separate IPython/notebook sessions.
%rerunRe-execute previous input lines, optionally filtered by range or pattern.
%dhistPrint the history of visited directories from %cd.

Auto-reload & Dev Workflow

%load_ext autoreloadLoad the autoreload extension (do this once per session).
%autoreload 2Automatically reload all modules before executing new code — great while editing local .py files.
%autoreload 0Disable autoreloading.
%autosave nSet the notebook's autosave interval in seconds.
%notebook fileExport the current session's input history to a notebook file.
%%prun -D out.profProfile a cell and dump the stats to a file for later inspection (e.g. with snakeviz).

Keyboard — Command Mode (press Esc first)

EnterSwitch into edit mode on the selected cell.
Shift + EnterRun the cell and select/create the next one.
Ctrl/Cmd + EnterRun the cell and keep it selected.
Alt/Option + EnterRun the cell and insert a new one below.
A / BInsert a new cell above / below the current one.
D, DDelete the selected cell (press D twice).
ZUndo cell deletion.
M / YConvert cell to Markdown / back to Code.
C / X / VCopy / cut / paste the selected cell.
Shift + MMerge the selected cell with the one below.
↑ / ↓ / J / KMove selection between cells.
LToggle line numbers for the selected cell.
Shift + LToggle line numbers for all cells.
I, IInterrupt the kernel (press I twice).
0, 0Restart the kernel (press 0 twice).
Shift + SpaceScroll the notebook up.

Keyboard — Edit Mode (cell has green border)

EscLeave edit mode and return to command mode.
TabTrigger code completion or indent, depending on cursor context.
Shift + TabShow the docstring / tooltip for the object under the cursor.
Ctrl/Cmd + /Toggle comment on the current line or selection.
Ctrl/Cmd + DDelete the entire current line.
Ctrl/Cmd + ZUndo the last text edit.
Ctrl/Cmd + Shift + -Split the cell at the cursor position.
Ctrl/Cmd + Home/EndJump to the start / end of the cell.
Ctrl/Cmd + ] / [Indent / dedent the current line or selection.
Commands vary slightly by IPython/Jupyter version and OS — use %lsmagic in your own kernel to confirm what's available.
