Python os Module Cheatsheet

Essentials & Imports

import osStandard import. For path-specific work also consider os.path or the more modern pathlib.Path.
os.nameName of the OS-dependent module: 'posix' (Linux/macOS) or 'nt' (Windows).
os.sepNative path separator — '/' on POSIX, '\\' on Windows.
os.linesepNative line terminator used when writing in text mode ('\n' or '\r\n').
os.extsep / os.pathsepExtension separator ('.') and the separator for path lists in PATH-style strings (':' or ';').
os.curdir / os.pardirStrings for the current directory ('.') and parent directory ('..').

Current Directory

os.getcwd()Return the current working directory as a string.
os.chdir(path)Change the current working directory to path.
os.fspath(path)Return the filesystem path representation of a path-like object.

Listing & Walking

os.listdir(path='.')Return a list of entry names in a directory (excludes . and ..).
os.scandir(path='.')Return an iterator of DirEntry objects — faster than listdir when you also need file type/stat info.
os.walk(top)Recursively generate (dirpath, dirnames, filenames) tuples for a directory tree.
os.walk(top, topdown=False)Walk the tree bottom-up instead of top-down — useful before recursive deletes.

Create & Remove

os.mkdir(path)Create a single directory. Raises FileExistsError if it already exists.
os.makedirs(path, exist_ok=False)Create a directory and any missing intermediate parents. Pass exist_ok=True to silently no-op if it exists.
os.rmdir(path)Remove a single, empty directory. Fails if it's not empty.
os.removedirs(path)Remove a directory and successive empty parent directories upward.
os.remove(path) / os.unlink(path)Delete a single file. Both are identical — unlink is the POSIX name.
os.rename(src, dst)Rename or move a file or directory from src to dst.
os.replace(src, dst)Like rename, but will silently overwrite dst if it already exists (cross-platform safe).
os.link(src, dst)Create a hard link pointing at src named dst.
os.symlink(src, dst)Create a symbolic link named dst pointing to src.
Note: os has no recursive "delete folder + contents" function — use shutil.rmtree() for that.

os.path — Inspecting Paths

os.path.join(a, b, ...)Join path components using the correct OS separator.
os.path.split(path)Split into (head, tail) — directory and final component.
os.path.basename(path)Return just the final filename component.
os.path.dirname(path)Return the directory portion of a path.
os.path.splitext(path)Split into (root, ext), e.g. ('photo', '.jpg').
os.path.abspath(path)Return an absolute, normalized version of a path.
os.path.realpath(path)Resolve symlinks and return the canonical absolute path.
os.path.normpath(path)Collapse redundant separators and up-level references (e.g. a/b/../c → a/c).
os.path.relpath(path, start)Return a relative path from start to path.
os.path.expanduser(path)Expand a leading ~ to the user's home directory.
os.path.expandvars(path)Expand $VAR / %VAR% environment variable references in a path string.

os.path — Existence & Type

os.path.exists(path)True if the path exists (file, dir, or broken-symlink target also counts as existing if it resolves).
os.path.isfile(path)True if the path exists and is a regular file.
os.path.isdir(path)True if the path exists and is a directory.
os.path.islink(path)True if the path is a symbolic link.
os.path.ismount(path)True if the path is a mount point.
os.path.isabs(path)True if the path is absolute rather than relative.
os.path.samefile(p1, p2)True if two paths refer to the same file on disk.
os.path.getsize(path)Return file size in bytes.
os.path.getmtime/getctime/getatime(path)Return modified / created / last-accessed timestamps (as epoch floats).

Permissions & Metadata

os.stat(path)Return a stat_result with size, timestamps, mode, owner UID/GID, etc.
os.lstat(path)Like stat(), but doesn't follow symlinks — reports on the link itself.
os.chmod(path, mode)Change file permissions, e.g. os.chmod(path, 0o755).
os.chown(path, uid, gid)Change file owner and group (POSIX only).
os.access(path, mode)Check real-user permissions, e.g. os.access(path, os.W_OK) for writability.
os.utime(path, times)Set a file's access and modified timestamps.
os.umask(mask)Set the process's umask and return the previous one.

Environment Variables

os.environDict-like mapping of the process's environment variables. Mutating it updates the env for child processes.
os.environ.get('VAR', default)Safely read an env var with a fallback if it's unset.
os.getenv('VAR', default)Equivalent shortcut to os.environ.get.
os.putenv(key, value)Set an environment variable at the OS level. Prefer os.environ[key] = value in practice.
os.unsetenv(key)Unset an environment variable at the OS level.

Processes & System Info

os.getpid()Return the current process's ID.
os.getppid()Return the parent process's ID (POSIX).
os.system(cmd)Run a shell command; returns the exit code. Output goes straight to stdout (not captured).
os.popen(cmd)Run a shell command and return a file-like object to read its output from.
os.kill(pid, sig)Send a signal to a process, e.g. os.kill(pid, signal.SIGTERM).
os.cpu_count()Return the number of CPUs available, or None if undetermined.
os.uname()Return system info (sysname, nodename, release, version, machine) — POSIX only.
os.getlogin()Return the name of the user logged in on the controlling terminal.
os.urandom(n)Return n random bytes suitable for cryptographic use.
os._exit(n)Immediately terminate the process without cleanup — rarely needed; prefer sys.exit().

Low-Level File Descriptors

os.open(path, flags)Open a file at the OS level, returning a raw file descriptor (int) rather than a file object.
os.read(fd, n)Read up to n bytes from file descriptor fd.
os.write(fd, data)Write bytes to file descriptor fd.
os.close(fd)Close a file descriptor opened with os.open.
os.dup(fd) / os.dup2(fd, fd2)Duplicate a file descriptor, optionally onto a specific target descriptor.
os.pipe()Create a pipe; return (read_fd, write_fd) for inter-process communication.
Note: for everyday file I/O, use the built-in open() instead — these are for low-level/POSIX-style work.
Many os functions are POSIX-only or behave differently on Windows — check the docs when writing cross-platform code.
For higher-level file operations (copy, move, recursive delete), see the shutil module · For object-oriented paths, see pathlib
