Python re — regex cheatsheet

Basic–intermediate reference for pattern matching in Python

Metacharacters — special meanings

CharacterMeaningExample
.Any single character (except newline by default)a.c matches abc, axc
^Start of string^hello matches only if string starts with hello
$End of stringend$ matches only if string ends with end
*Zero or more of the precedingab*c matches ac, abc, abbc
+One or more of the precedingab+c matches abc, abbc — not ac
?Zero or one of the preceding (also makes quantifiers lazy)colou?r matches color and colour
|OR — match either sidecat|dog matches cat or dog
( )Capture group(ab)+ matches ab, abab
[ ]Character class — match any one inside[aeiou] matches any vowel
[^ ]Negated class — match any one NOT inside[^0-9] matches any non-digit
\Escape — treat next char literally\. matches a real period

Quantifiers — how many times?

PatternMeaningLazy version
*0 or more*?
+1 or more+?
?0 or 1??
{n}Exactly n times—
{n,}n or more times{n,}?
{n,m}Between n and m times{n,m}?

Greedy means "match as much as possible." Lazy (? suffix) means "match as little as possible." Example: <.*> on <a>foo</a> matches the whole thing; <.*?> matches just <a>.

Flags

FlagShort formEffect
re.IGNORECASEre.ICase-insensitive matching
re.DOTALLre.SMake . match newlines too
re.MULTILINEre.M^ and $ match start/end of each line, not just the whole string
re.VERBOSEre.XAllow whitespace and comments in pattern for readability

Combine flags with |: e.g. re.search(pat, s, re.I | re.S)

re functions

FunctionReturnsUse when...
re.search(pat, s)First match object, or NoneYou just want to know if it matches anywhere in the string
re.match(pat, s)Match object if pattern matches at the start, or NoneYou want to match from the beginning only
re.fullmatch(pat, s)Match object if whole string matches, or NoneValidating input (e.g. "is this a valid date?")
re.findall(pat, s)List of all matches (strings)You want every occurrence
re.finditer(pat, s)Iterator of match objectsLike findall but you also need span/group info per match

re.split(pat, s)List of string piecesSplitting on a pattern (e.g. any whitespace)
re.compile(pat)Compiled pattern objectReusing the same pattern many times

If re.search() returns None (no match), calling .group() on it raises an AttributeError. Always check if m: before using the result.

Quick examples

GoalPattern / call
Extract a number(\d+)
Match an email (rough)[\w.+-]+@[\w-]+\.\w+
Text between two tags<tag[^>]*>(.*?)</tag> with re.DOTALL
A whole line^.*$ with re.MULTILINE
Replace whitespace runsre.sub(r'\s+', ' ', s)
Split on comma or semicolonre.split(r'[,;]\s*', s)
All words in a stringre.findall(r'\w+', s)
Named group example(?P<year>\d{4})-(?P<month>\d{2})

Always use raw strings for patterns: r'\d+' not '\d+'. Without the r, Python interprets backslash sequences before the regex engine ever sees them.
